# BlogChain — client

This is the Next.js frontend for BlogChain. It has no backend of its own —
it talks directly to a deployed Soroban smart contract via the Stellar RPC
and the Freighter wallet extension.

Project-wide documentation lives at the repo root, not here:

- [`../README.md`](../README.md) — overview, architecture, features
- [`../INSTALL.md`](../INSTALL.md) — install, run, build, deploy, troubleshooting
- [`../REQUIREMENTS.md`](../REQUIREMENTS.md) — required tool versions
- [`../DEPENDENCIES.md`](../DEPENDENCIES.md) — every dependency and what it's for

Quick start:

```bash
npm install
npm run dev
```
